# Oihana PHP Enums — Documentation

![Oihana PHP Enums](https://raw.githubusercontent.com/BcommeBois/oihana-php-enums/main/assets/images/oihana-php-enums-logo-inline-512x160.png)

[![Latest Version](https://img.shields.io/packagist/v/oihana/php-enums.svg?style=flat-square)](https://packagist.org/packages/oihana/php-enums)
[![License](https://img.shields.io/packagist/l/oihana/php-enums.svg?style=flat-square)](../LICENSE)

A PHP 8.4+ collection of **strongly-typed constant enumerations** — over **1,600 constants** across **73 classes** in five domains (general, HTTP, mail, JWT/JOSE, OAuth 2.0/OIDC), all sharing a single reflection API through `ConstantsTrait`. No *magic strings*.
