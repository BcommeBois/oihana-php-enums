Class IniOptions

This class defines a set of constants representing the names of PHP configuration directives (php.ini options).

These constants can be used to reference PHP ini settings in a consistent and typo-safe manner throughout the codebase.

It uses the ConstantsTrait to provide reflection utilities for retrieving the defined constants.
